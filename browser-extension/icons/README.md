# Extension Icons

These placeholder icon files should be replaced with proper PNG icons.

To generate from the SVG:
```bash
convert icon.svg -resize 16x16 icon16.png
convert icon.svg -resize 48x48 icon48.png
convert icon.svg -resize 128x128 icon128.png
```

Or use an online SVG to PNG converter:
- https://svgtopng.com/
- https://cloudconvert.com/svg-to-png

Upload `icon.svg` and generate 16x16, 48x48, and 128x128 PNG versions.
